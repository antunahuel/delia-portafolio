Descripción

En este proyecto, se desarrolla una API utilizando Node.js y varias dependencias gestionadas a través de Node Package Manager (npm). Las principales tecnologías y bibliotecas que se emplearán incluyen:

  • Express: Un framework minimalista y flexible para Node.js que facilita la creación de aplicaciones web y APIs.
  • Sequelize ORM: Un Object-Relational Mapping (ORM) para Node.js que permite interactuar con bases de datos SQL de manera sencilla y eficiente.
